# BuildSmart

A React-based construction management application designed to streamline building projects.

## Features

- Project management dashboard
- Resource allocation
- Timeline tracking
- Budget management
- Supplier coordination

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

```bash
# Clone the repository
git clone https://github.com/Warner-Ham/BuildSmart.git

# Navigate to the project directory
cd BuildSmart

# Install dependencies
npm install

# Start the development server
npm start
```
